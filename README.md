# react-rxjs

> A small library for creating applications based on unidirectional data flow with RxJS.

## Install

```bash
npm i -S react-rxjs
```

## Usage

```js
import ReactRxJS from 'react-rxjs';

export default ReactRxJS.RxInject(store$, propsFn)(MyComponent);
```
